# CodeAlfa Tasks
This repository contains Machine Learning tasks completed for CodeAlfa.

## Projects
- Car Price Prediction
- Iris Flower Classification

## Technologies Used
- Python
- pandas
- numpy
- scikit-learn

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Aratikadam/codealfa_tasks.git
   ```
2. Open the project folder.
3. Install required libraries:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Python file:
   ```bash
   python filename.py
   ```
