# CodeAlfa Tasks
This repository contains Data Science projects completed for the CodeAlfa internship.

## Projects Included
* **Car Price Prediction**
* **Iris Flower Classification**
* **Sales Prediction**
* **Unemployment Analysis**

## Technologies Used

Built using Python with popular data science libraries:
`pandas`, `numpy`, `scikit-learn` and others.

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Aratikadam/codealfa_tasks.git
   ```
2. Navigate to the project folder.
3. Install the dependencies:

   ```bash
   pip install -r requirements.txt
   ```
4. Run the Python scripts:

   ```bash
   python filename.py
   ```
